# 💪 BIGFIT Gym & Nutrition

Un site web moderne pour une salle de sport en Tunisie, développé avec React et Vite.

---

## 🚀 Aperçu

BIGFIT est une plateforme web conçue pour :
- Attirer de nouveaux clients
- Présenter les services fitness et nutrition
- Offrir une expérience utilisateur fluide et moderne

---

## 🛠️ Technologies utilisées

- ⚛️ React
- ⚡ Vite
- 🎨 CSS / Tailwind
- 🌐 Déploiement : Netlify

---

## ✨ Fonctionnalités

- Interface moderne et responsive 📱
- Sections organisées :
  - Hero (landing)
  - Programmes
  - Nutrition
  - Tarifs
  - Témoignages
  - Contact
- Design orienté conversion 💰
- Navigation fluide
